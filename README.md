## Project Theme
Our project is a **modern coffee shop website** designed to showcase products, allow users to browse the menu, place orders, and contact the shop. The site emphasizes a sleek, contemporary look with interactive features like live search, order modal, and cart functionality.

---

## Project Modules / Pages

| Module/Page Name        | Purpose/Functionality                                                                 | Responsible Member(s) |
|-------------------------|-------------------------------------------------------------------------------------|----------------------|
| Home / Hero Section     | Showcase brand, highlight featured drinks, slider for promotions                    | Nimra Tauf              |
| About                   | Provide information about the coffee shop, story, values                             | Nimra Rauf                 |
| Menu / Categories       | Display drink categories, list products with description, images, prices            | Nimra Rauf              |
| Live Search             | Allow users to search drinks by name or description                                  | Nimra Rauf                |
| Order / Modal           | Enable users to select a drink, view price, add to cart                              | Malaika Raheem                  |

| Features Section        | Highlight unique features, e.g., quality beans, fast delivery, cozy ambiance        | Malaika Raheem & Nimra Rauf               |
| Footer                  | Include social media links, address, copyright information                          | Nimra Rauf            |       

---

## Features
- Responsive design for mobile and desktop
- Interactive hero slider
- Live search for drinks
- Order modal with add-to-cart functionality
- Cart modal with quantity management
- Contact form with email validation
- Back-to-top navigation button
- Smooth scroll and hover animations
- Footer with social media and contact info
